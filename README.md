# FPGA Basketball Scoreboard (BASYS3, VHDL)

This project implements a basketball scoreboard using the BASYS3 FPGA board, VHDL, and external sensors.

## Features

- 60-second countdown timer
- Score tracking using motion sensor (+1)
- Magnetic field sensor scoring (+5)
- Four-digit seven-segment display
- Real-time hardware operation

## Hardware

- BASYS3 FPGA board
- Motion sensor
- Magnetic field sensor
- Seven-segment display
